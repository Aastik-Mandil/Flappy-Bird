\\# Flappy Bird



This project is about the Flappy Bird Game. It is a 2D game built on the \*\*Pygame\*\* library.





\\## Preview



Preview the example live on \\\[StackBlitz](http://stackblitz.com/):





\\## Tech Stack



Python, Pygame





\\## Run Locally



Clone the project



```bash

git clone https://github.com/Aastik-Mandil/Flappy-Bird.git

```





Go to the project directory



```bash

cd Flappy-Bird

```





Create virtual env



```bash

python -m venv .venv

```





Activate virtual env



```bash

cd .venv/Script

activate.bat

```





Install dependencies



```bash

pip install -r requirement.txt

```





Start the game



```bash

python -m main.py

```





De-activate virtual env



```bash

deactivate

```
